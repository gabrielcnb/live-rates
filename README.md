# Currency Converter Dashboard

Real-time currency dashboard showing USD, EUR, GBP exchange rates plus Bitcoin and Gold prices. No backend required — pure HTML/JS using free public APIs.

## Features

- Live exchange rates (USD, EUR, GBP, BRL)
- Bitcoin price in real-time
- Gold price per gram
- Percentage change indicators
- Auto-refresh every 30 seconds

## Stack

- HTML5 + CSS3 + Vanilla JavaScript
- [AwesomeAPI](https://docs.awesomeapi.com.br/) (free, no API key required)

## Usage

Just open `index.html` in any browser. No install needed.
